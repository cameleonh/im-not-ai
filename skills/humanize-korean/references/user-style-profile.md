# User Editorial Style Profile

Use this reference when the user asks to write or revise in "내 스타일", "Honey_hoony 스타일", "내가 쓴 글 참고", "칼럼처럼", or when the task is a short Korean argumentative essay.

Derived from the user's own essay screenshot titled "학계의 자정작용은 왜 작동하지 않는가?" The full text is not stored here. Use only the generalized style profile below.

## Core Shape

1. Start with a direct question title that names the institutional problem.
2. Open with the social norm or public expectation behind the issue.
3. Move quickly to a concrete recent case or visible failure.
4. Diagnose the institutional mechanism that made the failure rational.
5. Explain the deeper structural problem.
6. End with a clear institutional remedy and a normative closing sentence.

## Tone

- Critical but controlled.
- Public-interest oriented rather than personal.
- Institutional and structural, not anecdotal for its own sake.
- Plain Korean with strong judgment verbs.
- No humor, no ornament, no metaphor-heavy prose.

## Paragraph Pattern

- Each paragraph advances one argumentative move.
- Prefer 4-6 medium-length sentences per paragraph.
- Use rhetorical questions sparingly, mainly at the start of a diagnostic paragraph.
- Use transitions such as "그러나", "왜 이런 일이 벌어지는가?", "더 심각한 것은", "그 결과", "무엇보다", "만일".

## Sentence Pattern

- State the claim first, then explain the mechanism.
- Use compact cause-effect chains: "A가 흔들릴 때, B는 C할 수밖에 없다."
- Prefer direct institutional nouns: 사회, 학계, 정부 R&D, 테뉴어, 학자, 연구자, 평가권자, 보호 절차, 공개 검증.
- Use evaluative predicates clearly: "무겁다", "쓸쓸하다", "어려웠다", "변질되었다", "정비해야 한다", "증명해야 한다."

## Argument Moves

Use this sequence for similar essays:

1. Principle: why the institution is trusted.
2. Breach: what recent case shows that trust failed.
3. Mechanism: why the institution rewards silence or inaction.
4. Inversion: why the protected position fails to protect the vulnerable.
5. Reform: what rule, procedure, transparency, or protection should change.
6. Closing: if the institution cannot justify itself, public trust will erode.

## Do

- Keep proper nouns and concrete events if the user supplies them.
- Name who should act: 학회, 테뉴어트랙 교수, 정부, 평가기관, 연구자.
- Prefer "구조", "제도", "책임", "검증", "공개", "보호" when the topic is governance.
- Make the final sentence firm and memorable.

## Do Not

- Do not imitate private wording verbatim.
- Do not turn the prose into a newspaper editorial with excessive slogans.
- Do not add facts, accusations, or names not supplied by the user.
- Do not soften every judgment into hedging. The style allows clear normative claims.
